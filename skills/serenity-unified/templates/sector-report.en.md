# Sector Report Template

> An equity research report centered on this supply chain, not a single stock. Method internalized, revolves around the chain itself. Every ticker's market cap/status in the stack must be calibrated with primary-source real-time data.

---

## 1. Sector Verdict (Conclusion First, 1-2 Sentences)

Whether this chain is worth putting capital on + which cycle stage it's in + which layer the chokepoint sits at. Break it down by time dimension: "near-end already priced vs far-end not yet unlocked" — don't be vague with "already ran up / still has room."

---

## 2. First, Understand This Chain (Education Layer)

> Mandatory for non-specialist readers. Depth calibrated to reader expertise. For expert inquiries / users who already understand, compress to a single line.

Use plain language to explain: what this thing is, what old problem it solves, why it suddenly matters now, what segments are inside (who does optics / who does materials / who assembles / who delivers finished product). Don't assume the reader knows terms like "pluggable / substrate / epitaxy" — explain each in one line on first use.

---

---

## 4. Chokepoint Layer Positioning

Which layers are true single-point-of-failure / demand >> supply (identified chokepoints), and which layers are already fully priced.

---

## 5. Where the Remaining Asymmetry Is (Core Deliverable)

Rank by "market cap / TAM asymmetry × upstream degree × distance to ramp × institutional discovery status (undiscovered > already re-rated)" — point out layers/directions where residual asymmetry exists. Explicitly flag risks for tickers that have already surged or have weak fundamentals. Don't mechanically rank "most upstream = best."

**Judgment Made Visible (Mandatory)**: For each candidate layer/name identified, use plain language to call out which good-chokepoint criteria are hit / which red flags are triggered. Attempt ≥1 OSINT / multi-hop inference. No pseudo-scores or invented code words.

Do not output single-stock upside ranges — only give "which layers and which names to dig deeper on."

---

## 6. Catalysts / Leading Signals

- Raw material spot prices (e.g. 7N indium SMM ATH)
- Adjacent earnings / earnings call cross-read
- Major capex moves (investment / M&A / capacity lock-up)
- Industry conferences (GTC / OFC)
- Policy bills
- Index inclusion

Distinguish "will it happen" vs "when will it happen."

---

## 7. Red Flags / Falsification / Boundaries

- What would overturn the entire thesis
- Which layers are traps
- **This is a "residual asymmetry map," not a buy signal**: to actually build a position, run the single-stock report + review for individual tickers

---

## Hard Rules (Sector-Specific)

- "Undiscovered > already re-rated" ranking must be based on real-time price moves/valuation — cannot use static knowledge base as current state
- The review step must independently re-check key tickers' latest market caps/status (sector market cap snapshots go stale extremely fast)
- The deliverable is an objective spatial map by layer — don't pile on "who thinks what"
